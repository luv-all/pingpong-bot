# cam-preview

다중 웹캠을 **가로 한 창**으로 보는 프리뷰. CLI 없음.

`src/main.rs`의 `DEVICES`만 수정:

```rust
const DEVICES: &[i32] = &[0, 1];
```

패널마다 `WxH`, 측정 FPS, 드라이버 FPS, **FOURCC**(`MJPG`/`YUY2`)를 표시한다.

Arducam B0332는 열자마자 `MJPG` + `1280x800` + `120fps` + 짧은 노출을 요청한다.
(YUY2로 열리면 스펙상 ~10fps — 예전 15fps 근처가 그 케이스.)

| 키 | 동작 |
|----|------|
| `Space` | 동결 / 해제 |
| `q` / ESC | 종료 |

## 글로벌 셔터 vs 모션 블러

- **글로벌 셔터** = 롤링 왜곡(기울어진 직선) 방지. **노출 중 움직임 블러는 그대로**다.
- 블러를 줄이려면 **노출을 짧게** (밝기↑/게인↑ 트레이드오프) + 고FPS(MJPG).

## 글로벌 vs 롤링 확인

직선 물체를 가로로 빠르게 휘두른 뒤 Space로 동결 — 롤링이면 비스듬, 글로벌이면 직선(블러 줄무늬는 남을 수 있음).
